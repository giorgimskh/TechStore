# TechStore

TechStore is a modern, responsive e-commerce web application featuring a futuristic dark-mode aesthetic with interactive CSS animations, client-side state management, and live API integrations.

## Live Preview

- Live Demo: https://giorgimskh.github.io/TechStore/

## Features

- **Live Data Fetch**: Dynamically loads real consumer electronics from the Fake Store API (electronics category).
- **Responsive Layout**: Uses CSS Grid and Flexbox to deliver a polished, mobile-first responsive shopping experience.
- **Interactive Modals**: Features detail modals with zoomed image viewer overlays, active stock indicators, and price analytics.
- **Aesthetic Visuals**: Includes custom sharingan-style eye decorations and hover transitions.
- **Cart & History Management**: Interactive local state tracking for shopping cart additions, product comparison, and viewed history.
